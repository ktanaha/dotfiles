dotfiles
========

# READ ME

dotfile管理用のリポジトリ

# シンボリックリンク

```
ln -sv ~/dotfiles/<ファイル名> ~/
ln -sv ~/dotfiles/<ファイル名> <各種ツールのディレクトリ>
```
